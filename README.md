# LockBit-Ransomware-Campaign-Analysis-with-SOC-Detection-Recommendations

## Overview

This project analyzes the LockBit ransomware campaign from a defensive cybersecurity perspective. The goal is to understand attacker techniques, identify Indicators of Compromise (IOCs), and develop SOC detection strategies.

LockBit is a ransomware-as-a-service (RaaS) operation responsible for numerous attacks targeting enterprises worldwide.

This project focuses on mapping the attack chain, identifying detection opportunities, and creating SOC detection rules.

## Objectives

* Analyze LockBit ransomware campaign behavior
* Identify indicators of compromise (IOCs)
* Map attack techniques to MITRE ATT&CK
* Develop SOC detection rules
* Provide SIEM detection queries
* Create incident response recommendations

## Skills Demonstrated

* Threat Intelligence Analysis
* Malware Campaign Research
* MITRE ATT&CK Mapping
* SOC Detection Engineering
* SIEM Query Development
* Incident Response Planning

## Tools Used

* VirusTotal
* MITRE ATT&CK
* Sigma Rules
* YARA
* OSINT Threat Intelligence
* Splunk / Sentinel Queries
